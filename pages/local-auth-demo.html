How It Works

  • Local Storage: User accounts are stored in your browser's localStorage
  • No Email Verification: Users can login immediately after signup
  • No Database: Everything is stored locally in the browser
  • Session Management: Login state persists across browser sessions
  • Password Storage: Passwords are stored in plain text (demo only - use hashing in production)
  • Completely Offline: No internet connection required for authentication
